# Better JoP

Better JoP 是一个为 [Joy of Painting](https://www.curseforge.com/minecraft/mc-mods/joy-of-painting) 模组提供增强功能的客户端模组。当您从墙上取下画布进行修改时，它会在画布原位置显示一个半透明的实时投影，让您能够一边观察整体效果一边绘画，大幅提升大幅作品的创作体验。

## 特性

- **实时投影**：画布被破坏（取下）后，模组会记录其精确位置、朝向和旋转角度。当您再次手持该画布时，原位置会出现一个半透明的投影，内容与手中画布完全同步。
- **多投影支持**：可同时显示多个不同画布的投影（例如背包中有多幅画时）。
- **配置灵活**：
    - 投影移除延迟（游戏刻）
    - 画布检测范围：仅主手 / 主手和副手 / 整个背包
    - 投影数据持久化：退出服务器时自动保存投影信息，下次进入同一服务器时自动恢复
- **纯客户端**：所有效果仅自己可见，不影响服务器和其他玩家。
- **兼容性**：完美适配 Joy of Painting 的所有画布尺寸，并支持画布旋转。

## 依赖

- **必须安装**：
    - [Fabric Loader](https://fabricmc.net/) ≥0.14.0
    - [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
    - [Joy of Painting](https://www.curseforge.com/minecraft/mc-mods/joy-of-painting) ≥1.0.0
- **可选（强烈推荐）**：
    - [Cloth Config API](https://www.curseforge.com/minecraft/mc-mods/cloth-config) —— 用于配置界面
    - [Mod Menu](https://www.curseforge.com/minecraft/mc-mods/modmenu) —— 方便在游戏中打开配置

## 安装

1. 确保已安装上述必要模组。
2. 将下载的 `betterjop-X.X.X.jar` 放入客户端的 `mods` 文件夹。
3. 启动游戏，模组将自动生效。

## 配置

如果您安装了 Cloth Config API 和 Mod Menu，可以在游戏内 Mod Menu 界面点击 Better JoP 的配置按钮进行设置。配置项包括：

- **投影移除延迟**（游戏刻）  
  当手中不再持有画布时，等待多少刻后移除投影（20 刻 = 1 秒）。默认值 15，可调范围 0~100。

- **画布检测范围**
    - 仅主手
    - 主手和副手（默认）
    - 整个背包

- **启用投影数据持久化**  
  退出服务器时自动保存所有投影信息，下次进入同一服务器时自动恢复。默认开启。

## 使用方法

1. 在墙上放置一幅画（任何尺寸均可），并绘制一些内容。
2. 直接左键击打画布（破坏实体），画布会掉落为物品。
3. 手持该画布（主手或副手，取决于您的检测范围配置），原位置会出现一个半透明的投影，显示画布当前内容。
4. 您可以手持画布正常打开绘画界面进行修改，投影会实时更新。
5. 当您将画布放回墙上（右键点击放置），投影自动消失。
6. 如果您启用了持久化，退出服务器后下次进入，之前记录的投影位置仍会保留，只要您再次手持对应画布，投影就会恢复。

## 注意事项

- 本模组为**纯客户端模组**，仅需安装在客户端。在多人服务器中游玩时，其他玩家**不会**看到您的投影。

- 如果使用全背包检测模式，请注意背包中可能有多个不同名称的画布，它们各自的投影会同时显示。
- 持久化功能仅对多人服务器有效，单人游戏不会保存投影信息。
